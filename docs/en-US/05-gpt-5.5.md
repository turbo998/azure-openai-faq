# 05 · GPT-5.5: OpenAI vs Azure OpenAI

Same capabilities. Different invocation details.

## 1. Entry point

### OpenAI
```python
from openai import OpenAI
client = OpenAI()
resp = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "tell me a joke"}],
)
```

### Azure OpenAI
```python
from openai import AzureOpenAI
client = AzureOpenAI(
    azure_endpoint=os.environ["AZURE_OPENAI_ENDPOINT"],
    api_key=os.environ["AZURE_OPENAI_API_KEY"],
    api_version=os.environ["AZURE_OPENAI_API_VERSION"],
)
resp = client.chat.completions.create(
    model=os.environ["AZURE_OPENAI_GPT55_DEPLOYMENT"],   # deployment name!
    messages=[{"role": "user", "content": "tell me a joke"}],
)
```

## 2. `reasoning_effort`
```python
resp = client.chat.completions.create(
    model=MODEL_OR_DEPLOYMENT,
    reasoning_effort="medium",        # minimal | low | medium | high
    messages=[...],
)
```
Same param both sides. Azure needs a recent `api-version` (preview is fine).

## 3. Responses API
```python
resp = client.responses.create(model=MODEL_OR_DEPLOYMENT, input="summarize relativity in 3 sentences")
print(resp.output_text)
```

## 4. Streaming
```python
stream = client.chat.completions.create(model=MODEL_OR_DEPLOYMENT, messages=[...], stream=True)
for chunk in stream:
    if not chunk.choices: continue          # Azure may emit filter-only chunks
    print(chunk.choices[0].delta.content or "", end="", flush=True)
```

## 5. Content filter (Azure)
On RAI block: HTTP 400 with `error.code = "content_filter"`. Inspect `error.innererror.content_filter_result` for category severity.

## 6. Quota
- OpenAI: account-level RPM/TPM.
- Azure: **per-deployment** TPM/RPM/PTU. Split deployments to spread quota.

## 7. Common error mapping
| Error | OpenAI meaning | Azure meaning |
|---|---|---|
| `model_not_found` | wrong model id / no access | wrong **deployment** name / model not deployed |
| `unsupported_parameter` | old SDK | old SDK or old `api-version` |
| `content_filter` | — | RAI block; tune severity or rephrase |
