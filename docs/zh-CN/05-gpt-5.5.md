# 05 · GPT-5.5 在 OpenAI 与 Azure OpenAI 上的差异

GPT-5.5 在两边能力对齐，但调用细节不同。

## 1. 调用入口

### OpenAI
```python
from openai import OpenAI
client = OpenAI()
resp = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "讲一个冷笑话"}],
)
print(resp.choices[0].message.content)
```

### Azure OpenAI
```python
from openai import AzureOpenAI
client = AzureOpenAI(
    azure_endpoint=os.environ["AZURE_OPENAI_ENDPOINT"],
    api_key=os.environ["AZURE_OPENAI_API_KEY"],
    api_version=os.environ["AZURE_OPENAI_API_VERSION"],   # e.g. "preview"
)
resp = client.chat.completions.create(
    model=os.environ["AZURE_OPENAI_GPT55_DEPLOYMENT"],   # 部署名！
    messages=[{"role": "user", "content": "讲一个冷笑话"}],
)
```

## 2. Reasoning effort
GPT-5.5 支持 `reasoning_effort` 控制思考强度：
```python
resp = client.chat.completions.create(
    model=MODEL_OR_DEPLOYMENT,
    reasoning_effort="medium",          # "minimal" | "low" | "medium" | "high"
    messages=[{"role": "user", "content": "证明费马小定理"}],
)
```
两边参数名一致；Azure 上需要相应 `api-version`（`preview` 通常即可）。

## 3. Responses API
Microsoft Foundry 与 OpenAI 都支持 `client.responses.create(...)`，是 chat completions 的下一代抽象（统一 tools、streaming、stateful）。
```python
resp = client.responses.create(
    model=MODEL_OR_DEPLOYMENT,
    input="用三句话总结相对论",
)
print(resp.output_text)
```

## 4. 流式
```python
stream = client.chat.completions.create(
    model=MODEL_OR_DEPLOYMENT,
    messages=[...],
    stream=True,
)
for chunk in stream:
    delta = chunk.choices[0].delta.content or ""
    print(delta, end="", flush=True)
```
**Azure 流式响应的每个 chunk 还会带 `prompt_filter_results` / `content_filter_results`** —— 业务代码应做空值兜底，遍历 `chunk.choices` 前先 `if chunk.choices:`。

## 5. 内容过滤差异
Azure 上若被 RAI 拦截会返回 `400` 加 `content_filter` error code，附带每类（hate/violence/sexual/self_harm/jailbreak）的 severity。OpenAI 端通常不在 chat 直接拦，需自行调 Moderations。

## 6. 速率限制 / 配额
- OpenAI：账户级 RPM/TPM。
- Azure：**deployment 级** TPM/RPM/PTU。可以为同一模型创建多个 deployment 拆配额。

## 7. 兼容代码片段
```python
def chat(prompt: str) -> str:
    client, model = make_client()   # 见 04 章
    resp = client.chat.completions.create(
        model=model,
        reasoning_effort="medium",
        messages=[{"role": "user", "content": prompt}],
    )
    return resp.choices[0].message.content
```

## 8. 常见报错对照
| 报错 | OpenAI 含义 | Azure 含义 |
|---|---|---|
| `model_not_found` | 模型名错 / 账户没访问权 | **deployment 名** 错 / 资源未部署该模型 |
| `unsupported_parameter` | SDK 版本旧 | SDK 版本旧或 `api-version` 旧 |
| `content_filter` | — | RAI 拦截，调 severity 阈值或改提示 |
